Arquivo: views/admin/imoveis.ejs
Tipo: HTML5/EJS
Descrição:
    O arquivo deve implementar uma tabela com informações de todos os imoveis
    além de botões para editar e deletar cada entrada da 
    tabela. A página é renderizada com um parâmetro  'imoveis' disponível 
    que é um array contendo todos os imoveis do sistema.
    A rota para editar um imovel é /admin/imoveis/search/:id
    A rota para deletar um imovel é /admin/imoveis/delete/:id

    A página deve também linkar para a página de criação de um novo imovel
    em /admin/imoveis/new
